# IRLTSPICE - A Senior Design Project
 > *Zach DeNeve, Parker Hitchcock, Clayton Hughes, Micah Samuel*

## Overview

LTspice + irl = IRLTSPICE! IRLTSPICE is a configuratble analog circuit.

## Project Structure
The senior design project uses a monorepo architecture with the following sub-directories:
- `hardware/` Hardware schematics
- `firmware/` Microcontroller firmware
- `software/` On-PC software
